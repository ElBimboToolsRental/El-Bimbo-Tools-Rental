# El Bimbo Tools Rental

Ready-to-upload GitHub Pages website package.

## Files
- `index.html` — main website
- `logo.png` — business logo
- `cover.png` — hero cover image

## How to deploy
1. Create a new GitHub repository
2. Upload all files in this folder
3. Go to **Settings > Pages**
4. Under **Build and deployment**, choose **Deploy from a branch**
5. Select **main** branch and **/root**
6. Save

Your site will be live at:
`https://YOUR-USERNAME.github.io/REPOSITORY-NAME/`

## Important
- Replace the Facebook link in `index.html` with your real Facebook page URL
- You can update prices and packages directly inside `index.html`
